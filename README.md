Bu depo, C programlama dersleri için hazırlanmış örnek projeleri ve uygulamaları içerir.  
Tüm dersler **basitten karmaşığa** doğru sıralanmıştır ve her ders kendi klasöründe **bin** ve **src** alt klasörleri ile organize edilmiştir.

---

## 📂 Ders Yapısı

| Klasör | İçerik |
|--------|--------|
| **01_ilkProgram** | İlk program ve temel yapı |
| **02_temelTipler** | Veri tipleri (int, char, double, float vb.) |
| **03_operatorler** | Atama, karşılaştırma, mantıksal ve aritmetik operatörler |
| **04_ifKosullari** | if, else ve koşul ifadeleri |
| **05_switchCase** | switch-case yapıları |
| **06_forDongusu** | for döngüsü örnekleri |
| **07_whileDongusu** | while ve do-while döngüleri |
| **08_diziler** | Dizi kullanımı ve fonksiyonlarla diziler |
| **09_fonksiyonlar** | Fonksiyon tanımlama, parametre ve dönüş değerleri |
| **10_recursion** | Rekürsif (özyinelemeli) fonksiyonlar |
| **11_matematikProblemleri** | Matematik problemleri ve algoritmalar |
| **12_uygulamalar** | Küçük uygulama örnekleri (hesaplamalar, şekiller vb.) |

> Not: Her klasörde `src` kaynak dosyaları, `bin` derlenmiş çalıştırılabilir dosyaları içerir.

---

## 🚀 Nasıl Kullanılır?

1. `src` klasöründeki `.c` dosyalarını derleyip çalıştırabilirsiniz:  

```bash
# Örnek: İlk Program
gcc src/ilkProgram.c -o bin/ilkProgram
./bin/ilkProgram
```
2.'bin' klasöründe derlenmiş dosyaları doğrudan çalıştırabilirsiniz:  

```bash
# Örnek: İlk Program
./bin/ilkProgram
```
